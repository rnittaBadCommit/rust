# the_book

このディレクトリは、Rust 公式ドキュメント The Book を順に進めるための作業場所です。
章ごとにコード、メモ、試したことを分けて整理していきます。

## 目的

- The Book の内容を、読むだけでなく手を動かして確認する
- 必要に応じて C / C++ と比較しながら理解を固める
- その場の一時メモではなく、あとで読み返せる形で残す

## 進め方

1. 対象の章を読む
2. 章で出てきたコードを手元で動かす
3. 詰まった点や気づいた点を章ごとのメモに残す
4. 必要なら、このリポジトリの `knowledge/` に再利用価値のある内容を整理する

## 置き方

- 章ごとにディレクトリを作る
- 小さなメモは各章ディレクトリの `notes.md` に置く
- 実行するコードは必要に応じて Cargo project として作る

例:

```text
the_book/
  README.md
  ch02_guessing_game/
    notes.md
    guessing_game/
```

## メモ

- 説明は日本語を基本にする
- Rust 特有の概念は、まず C で言い換えられないか考える
- C だと不自然なら C++ と比較してよい
- 抽象論より、小さく実行できる例を優先する
