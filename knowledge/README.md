## Index

## Rule

- 質問、レビュー、コード修正から得られた再利用価値のある知識を残す
- 1 ファイル 1 テーマを基本にする
- 整理しやすい場合は、サブディレクトリを作ってよい
- 後から読み返しやすいように、必要に応じて既存ファイルを分割・統合する
- まだ十分に身についていない知識は、明示指示がある場合のみ `not_yet_internalized/` に保存する
