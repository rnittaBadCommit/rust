## Learned

- `for x in xs`, `for x in &xs`, `xs.iter()` の違いは、C の「値を動かすのか、ポインタで借りるのか」と対比すると読みやすい
- `where` 句、associated type、`use` のような構文は、意味ごとに分けてノート化すると混ざりにくい
- `&[T]` と `&[T; N]` の違いは「fat pointer かどうか」を意識すると整理しやすい
- trait object は C の関数ポインタ表に近い見方で入り、`trait bound` と対比すると役割を切り分けやすい
- `ex01` の `strlen`, `max_option`, `stack`, `line_count`, `value_enum` を進めると、所有権と enum の基礎がまとめて復習できる

## Questions
