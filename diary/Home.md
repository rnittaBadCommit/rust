## Workflow

1. 学習した日は `daily/` に日記を書く
2. まとまった概念は `topics/` に整理する
3. 再利用したいコードは `snippets/` に残す
4. 新規ノートは `templates/` のテンプレートから作る

## First Targets

- Ownership / Borrowing
- `String` と `&str`
- `Result<T, E>` と `?`
- Generics
- Traits basics
- `Add`, `AddAssign`, `Deref`

## Current Blockers

- trait の意味と役割がまだ曖昧
- `impl Trait for Type` の読み方に慣れていない
- `Add`, `AddAssign`, `Deref` が trait の具体例としてまだ結び付いていない
